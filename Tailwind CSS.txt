Tailwind CSS (utility classes)
------------
.tailwindcss.com
.flow bite

. By using this, no need to explicitly write css.

. Tailwind css intelliSense
. <script src="https://cdn.tailwindcss.com"></script>

. npm init
. npm install -D tailwindcss postcss autoprefixer
  npm install vite
. In package.json,
  "scripts":{
     "start": "vite"
  },
. index.css  ->  @tailwind base;  @tailwind components;  @tailwind utilities; 
  link this css file to html file.
. npx tailwindcss init    
. tailwind.config.js -> content["*"] 

Why use Tailwind
----------------
. No re-inventing of class names required.
. Your CSS doesn't grow with html and designs
. When you make a change, no risk of breaking existing templates.
. Does it make sites slow? No
  The classes are shipped which are used

Apply directive
---------------
When we want to design a button with our desired styles and use it like sm,lg,xl to get small, large and xlarge button.

style.css ->
 
.btn{
   @apply bg-yellow-100 mx-2 my-4 text-sm;
}

use it as   class="btn"

Layer directive
---------------
We can stick these styles inside layers like base, components, utilities.

@tailwind base;
@tailwind components;
.componentsafter{
    text-align:center;
}
@tailwind utilities;

@layer components{
  .btn{
      @apply bg-red-300 mx-2 my-4;
  }
}    

extend

. When some specifications are not available, we use this

   spacing:{
       13 : '3.25rem'
   }
}
